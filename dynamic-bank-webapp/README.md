# Dynamic Bank - WebApp

This is the web frontend project of Dynamic Bank.
It simulates a simple banking application.

The webapp is built with Spring MVC + Thymeleaf and communicates with the Account Service and Transfer Service microservices via REST APIs.

---

## Tech Stack:

- **Java 17**
- **Spring Boot 4.0.0**
- **Spring MVC + Thymeleaf**
- **Spring Web**
- **Spring Validation (Hibernate Validator)**
- **Spring Session + Redis** (for session management)
- **Spring Security** (for access control and authentication)
- **Spring Boot Actuator** (for observability and metrics)
- **Maven** (dependency management)
- **PostgreSQL, Kafka and Redis** integrated via microservices
